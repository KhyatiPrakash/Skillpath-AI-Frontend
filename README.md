# SkillPath AI - Frontend

## 📌 Overview

SkillPath AI is a career guidance web application developed using React and the MERN stack. It enables users to explore careers, save career paths, and manage their profile through a clean and responsive interface.

---

## ✨ Features

- Responsive Landing Page
- User Authentication
- Dashboard
- Career Listing
- Career Details Page
- Search Careers
- Filter Careers
- Save Careers
- Admin Dashboard
- Add Career
- Edit Career
- Delete Career
- Protected Routes

---

## 🛠 Tech Stack

- React
- React Router
- Axios
- CSS
- Context API

## ⚙️ Installation

Clone the repository

```bash
git clone <frontend-repository-url>
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```
